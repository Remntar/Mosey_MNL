Preguntas Frecuentes (FAQ) General 1. ¿Qué es esta aplicación de películas? Nuestra aplicación de películas te permite explorar una vasta base de datos de películas, incluyendo detalles como sinopsis, actores, trailers y reseñas. Toda la información se obtiene a través de una API de películas. 2. ¿Cómo funciona la aplicación? La aplicación se conecta a una API de películas para recuperar información sobre películas, incluyendo las más recientes, las más populares y las mejor calificadas. Los usuarios pueden buscar películas, ver detalles específicos y guardar sus favoritas. Cuenta y Configuración 3. ¿Necesito crear una cuenta para usar la aplicación? No, no es necesario crear una cuenta para explorar las películas. Sin embargo, si deseas guardar tus películas favoritas o escribir reseñas, necesitarás registrarte. 4. ¿Cómo puedo crear una cuenta? Puedes crear una cuenta proporcionando tu dirección de correo electrónico y creando una contraseña, o registrándote a través de tus cuentas de Google o Facebook. 5. Olvidé mi contraseña, ¿cómo puedo restablecerla? En la pantalla de inicio de sesión, haz clic en "¿Olvidaste tu contraseña?" y sigue las instrucciones para restablecerla a través de tu correo electrónico. Uso de la Aplicación 6. ¿Cómo busco una película específica? Utiliza la barra de búsqueda en la parte superior de la pantalla principal. Escribe el nombre de la película y selecciona el resultado adecuado de la lista desplegable. 7. ¿Cómo puedo ver los detalles de una película? Haz clic en el título de la película de la lista de resultados de búsqueda o de las listas de películas destacadas para ver detalles como la sinopsis, el elenco, el director, las reseñas y el trailer. 8. ¿Puedo guardar películas para ver más tarde? Sí, puedes agregar películas a tu lista de favoritos haciendo clic en el icono de corazón en la página de detalles de la película. Necesitarás estar registrado y haber iniciado sesión para usar esta función. **9. ¿Cómo puedo escribir una reseña?** Ve a la página de detalles de la película que deseas reseñar y haz clic en "Escribir reseña". Necesitarás estar registrado y haber iniciado sesión para publicar tu reseña. ### Problemas Técnicos **10. La aplicación no está funcionando correctamente, ¿qué puedo hacer?** Primero, intenta cerrar y volver a abrir la aplicación. Si el problema persiste, verifica tu conexión a internet. Si aún tienes problemas, ponte en contacto con nuestro equipo de soporte a través del correo [soporte@tudominio.com](mailto:soporte@tudominio.com). **11. No puedo encontrar una película específica, ¿por qué?** Es posible que la película no esté en la base de datos de la API que estamos utilizando. Si crees que debería estar, por favor, contáctanos y lo investigaremos. ### Seguridad y Privacidad **12. ¿Qué información personal recopila la aplicación?** Recopilamos solo la información necesaria para crear y gestionar tu cuenta, como tu dirección de correo electrónico. No compartimos tu información personal con terceros sin tu consentimiento. **13. ¿Cómo puedo eliminar mi cuenta?** Para eliminar tu cuenta, ve a la configuración de tu perfil y selecciona "Eliminar cuenta". Esta acción es irreversible y todos tus datos se borrarán permanentemente. ### API y Datos **14. ¿Qué API de películas utiliza esta aplicación?** Utilizamos la API de [Nombre de la API], que proporciona información actualizada y precisa sobre películas de todo el mundo. **15. ¿Con qué frecuencia se actualizan los datos de las películas?** Los datos se actualizan en tiempo real a medida que la API recibe nueva información. ---
